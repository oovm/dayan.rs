接下来有个分歧点, 接下来的 $1$ 应该加在哪?

常规想法是令:

$$
p(1, 1, 2)  = ε_0 + 1
$$

如果是按照 Veblen 规则, 则还需经过

$$
\begin{aligned}
v(1, v(1, 1, 1)) &= ε_0 + 1\\
v(v(1, 1, 1), 1) &= {ε_0}^{ε_0}\\
v(1, 1, 2)  &= ε_1\\
\end{aligned}
$$

简单起见, 我们希望内层不要比外层大, 采用第一种方案继续分析.

| Alpha                                                      | Ordinal     |

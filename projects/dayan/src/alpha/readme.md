此时定义:

$$
p(1, 1, 1)  = ε_0
$$

常规想法是令:

$$
p(1, 1, 2)  = ε_0 + 1
$$

但是还有一种强度更高的定义方法:

$$
\begin{aligned}
f(1, f(1, 1, 1)) &= ε_0 + 1\\
f(f(1, 1, 1), 1) &= {ε_0}^{ε_0}\\
f(1, 1, 2)  &= ε_1\\
\end{aligned}
$$
